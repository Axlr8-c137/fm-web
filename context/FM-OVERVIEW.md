

**1\. PROJECT OVERVIEW** 

This quotation covers the end-to-end design, development, testing, and deployment of a custom Facility Management Application: an Android mobile app for field employees, client-facing attendance reporting, an operations monitoring dashboard, and a full payroll web portal with Indian statutory compliance (PF, ESIC, TDS, PT). Designed for 1,000 \- 3,000 active users at launch. 

**2\. V1 SCOPE \- INCLUDED IN THIS QUOTE**

| Module  | Features Included  | Ver. |
| ----- | :---- | :---- |
| **Employee Attendance & Geofencing** | Mobile OTP login, geofence-based punch in/out, facial recognition, shift schedules, task view, push notifications, Employee ID tab | **V1** |
| **Site Updates**  | Real-time site status reporting, text \+ photo \+ video attachments, timestamped per employee per site | **V1** |
| **Payroll Web Portal (Full Engine)** | Admin login, salary management, PF/ESIC/TDS/PT computation, payroll runs, bank file generation, payroll reports | **V1** |
| **Client Attendance  Reporting** | Monthly, weekly, daily attendance reports for client-assigned personnel, late arrivals, absences  | **V1** |
| **Operations Monitoring Dashboard** | Real-time employee tracking, geofence exit alerts (\>5 min), device battery/signal/airplane mode audit, heatmap, employee onboarding (Aadhaar, PAN, ration card, police verification, PF & ESIC docs) | **V1** |
| **Super Admin Panel**  | Full system access, user/role management, site CRUD, configuration, data export, product-owner control | **V1** |
| **Multi-language Support**  | English, Hindi, Marathi across all V1 mobile modules  | **V1** |
| **Backend &  Infrastructure** | Java 17 / Spring Boot 3.4 API, PostgreSQL \+ PostGIS, Redis, NGINX, object storage (R2/S3), JWT auth, HTTPS, push notifications, Spring Scheduler background jobs, 30-day data retention | **V1** |
